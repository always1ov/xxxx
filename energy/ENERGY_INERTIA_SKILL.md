# 能量惯性指标 Skill

基于用户自定义通达信公式，通过量价合力的双重平滑，识别能量翻转买点与多空能量强度。

## 公式原文（不得修改任何逻辑与字符）

```
能量:=SQRT(VOL)*(((CLOSE-(HIGH+LOW)/2))/((HIGH+LOW)/2));
平滑能量:=EMA(能量,16);
能量惯性:EMA(平滑能量,16);
DRAWICON(能量惯性>0 AND REF(能量惯性,1)<0,0,1);
STICKLINE(能量惯性>=0,(能量惯性-能量惯性*0.05),(能量惯性-能量惯性*0.15),3,0),COLOR0000CC;
STICKLINE(能量惯性>=0,(能量惯性-能量惯性*0.2),(能量惯性-能量惯性*0.35),3,0),COLOR0066FF;
STICKLINE(能量惯性>=0,(能量惯性-能量惯性*0.4),(能量惯性-能量惯性*0.55),3,0),COLOR0099FF;

```

## 数据来源

使用 `tdx_get_kline` 获取日 K 线数据：

```
tdx_get_kline(code="sh600519", period="day", adjust="qfq", count=250)
```

所需字段：`close`（收盘价）、`high`（最高价）、`low`（最低价）、`volume`（成交量，单位：手）

---

## 计算步骤（严格按公式逻辑，不得改动）

### EMA 计算函数

```
EMA[0] = series[0]
EMA[i] = EMA[i-1] * (n-1)/(n+1) + series[i] * 2/(n+1)
```

### 第一步：能量

```
能量 = SQRT(VOL) * ((CLOSE - (HIGH + LOW) / 2) / ((HIGH + LOW) / 2))
```

含义：成交量开方作为权重，乘以收盘价偏离高低均值的幅度。
- 收盘价高于中枢 → 能量为正（多头）
- 收盘价低于中枢 → 能量为负（空头）

### 第二步：平滑能量

```
平滑能量 = EMA(能量, 16)
```

### 第三步：能量惯性

```
能量惯性 = EMA(平滑能量, 16)
```

双重 EMA 平滑，过滤短期噪音，反映中期量价合力方向。

### 第四步：翻转信号

```
翻转买点：能量惯性 > 0 AND REF(能量惯性, 1) < 0
即：当前能量惯性由负转正的第一根K线
```

---

## 信号解读规则

| 状态 | 条件 | 含义 |
|------|------|------|
| 翻转买点 🔵 | 能量惯性由负转正 | 能量方向切换，潜在趋势启动 |
| 多头能量 | 能量惯性 > 0 且持续上升 | 量价合力偏多，趋势向上 |
| 多头能量减弱 | 能量惯性 > 0 且持续下降 | 多头动能衰减，注意转折 |
| 空头能量 | 能量惯性 < 0 且持续下降 | 量价合力偏空，趋势向下 |
| 空头能量减弱 | 能量惯性 < 0 且持续上升 | 空头动能衰减，关注翻转 |

## 输出格式

```
【能量惯性】
前日：{值}（多头/空头，上升/下降）
昨日：{值}（多头/空头，上升/下降）
当前：{值}（多头/空头，上升/下降）{翻转买点 🔵}
解读：{信号解读}
```

---

## 注意事项

- 需要 close、high、low、volume 四个字段，建议取 250 根以上预热
- 能量惯性为中期指标，对短线噪音不敏感，适合判断趋势方向
- 翻转买点需结合其他信号确认，单独使用存在假突破风险
- 公式逻辑与字符不得修改
