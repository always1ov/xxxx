# SKILL: 能量惯性计算

## 依赖数据
- `tdx_get_kline`（period="day", adjust="qfq", count=500）
- 字段：CLOSE, HIGH, LOW, VOL

## 计算步骤

### 第一步：能量序列

```
能量    = SQRT(VOL) * ((CLOSE - (HIGH + LOW) / 2) / ((HIGH + LOW) / 2))
平滑能量 = EMA(能量, 16)
能量惯性 = EMA(平滑能量, 16)
```

> 说明：能量 > 0 表示收盘偏强（高于中间价），能量 < 0 表示收盘偏弱（低于中间价），以成交量开方加权放大差异。

### 第二步：状态判定

```
多头    = 能量惯性 > 0
空头    = 能量惯性 < 0
上升    = 能量惯性 > REF(能量惯性, 1)
下降    = 能量惯性 < REF(能量惯性, 1)
翻转买点 = 能量惯性 > 0 AND REF(能量惯性, 1) < 0   // 由负转正的第一根
```

### 第三步：状态组合解读

| 状态 | 解读 |
|------|------|
| 多头 + 上升 | 多头能量增强，趋势向上 |
| 多头 + 下降 | 多头动能衰减，注意转折 |
| 空头 + 下降 | 空头能量增强，趋势向下，回避 |
| 空头 + 上升 | 空头动能衰减，关注翻转 |
| 🔵 翻转买点 | 能量由负转正第一根，潜在趋势启动，关注确认 |
